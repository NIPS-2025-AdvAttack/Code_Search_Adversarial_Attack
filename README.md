# Code Search Adversarial Attack


Effective retrieval of code snippets using natural language queries is crucial for enhancing code reuse and development efficiency in software engineering. However, current neural code language models (CLMs) often rely on superficial textual features, such as identifiers, rather than the underlying code functionality for the code search task. In this paper, we introduce a language-agnostic adversarial attack method targeting these vulnerabilities in neural CLMs. Our approach modifies identifiers within a code snippet to deceptively increase its similarity to an arbitrary target query in the embedding space. We also demonstrate that adversarial code snippets generated using small models exhibit significant transferability, effectively degrading the performance of 50 times larger models or closed-source systems. Experiments confirm substantial performance drops across various models and benchmarks under this attack transfer. These findings highlight the fragility of current code search methods and underscore the need for more robust, logic-aware approaches.
